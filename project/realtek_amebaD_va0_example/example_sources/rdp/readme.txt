Example Description

This example describes how to build and use RDP.

The steps to build RDP image are as following:
(1) Copy 'main.c' to realtek_amebaD_va0_example\src\src_hp
(2) Copy 'secure_src.c' to component\soc\realtek\amebad\img3 and override the original one.
(3) rebuild project

The steps to configure RDP and download RDP image can be found in "Chapter RDP, AN0400 Ameba-D Application Note.pfd".
