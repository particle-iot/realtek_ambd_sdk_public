Example Description

This example describes how to use ADC normal channel.

1. The corresponding pin of each channel:
	channel_idx	pin
	ADC_CH0 	PB_4
	ADC_CH1		PB_5
	ADC_CH2		PB_6
	ADC_CH3		PB_7
	ADC_CH4		PB_1
	ADC_CH5		PB_2
	ADC_CH6		PB_3
	ADC_CH7		VBAT_MEAS
	   
2. ADC_CH7(VBAT) calibration parameters(gain & offset) are different with normal channel
